Chromium Example User Interface

(Web UI wrapped in a native macOS Application)

Demonstrates HTML/Javascript control & interaction with the App

Some things you can do from this UI, for example:
  • If you just simulated a login, Get that Authentication Data from the App side (e.g. from Web Service).

  • Make the wrapping App perform an action, for instance Go to its Next Screen.
Control resources that the app-side has exclusive access to, like the file system...
  • Click here to open an arbitrary file (e.g. this HTML file) on your local system:
    Open this HTML File

  • Simulate the process of uploading a file, by popping-up a File Picker:
    Upload File

  • Upload some so-called "Special" File
Or simply Log Out
